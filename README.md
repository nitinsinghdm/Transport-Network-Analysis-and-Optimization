# Public Transport Optimization using AI & Big Data

## Overview

This project analyzes public transport systems at two levels:

* **Germany (Macro Analysis)** – Rail network structure and performance
* **Berlin (Micro Analysis)** – Multi-modal urban transport dynamics

## Key Components

* Travel time distribution analysis
* Service frequency and network hierarchy
* Station connectivity and centrality
* Mode-wise performance evaluation

## Next Step

Building a **multi-modal journey simulation system** to evaluate:

* Travel time
* Transfers
* Delay risk
* Route alternatives

## Tech Stack

* Python (Pandas, Seaborn)
* GTFS Data
* Jupyter Notebooks
